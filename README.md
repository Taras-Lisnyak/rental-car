# RentalCar Frontend

Frontend‑частина застосунку для сервісу оренди автомобілів. Реалізовано на
**Next.js** з використанням **TypeScript** та модульних CSS‑стилів.

## 🚀 Основні можливості

- Головна сторінка з банером та навігацією
- Каталог автомобілів з пагінацією
- Сторінка деталей авто (CarDetails) з інформацією та формою бронювання
- Адаптивний дизайн, стилі на основі макету
- Синхронізація теми та локальних налаштувань

## 🛠️ Технології

- [Next.js](https://nextjs.org/) — фреймворк для React з підтримкою SSR
- [TypeScript](https://www.typescriptlang.org/) — статична типізація
- CSS Modules — ізольовані стилі для компонентів

## ⚙️ Запуск проєкту

```bash
# Клонування репозиторію
git clone

# Перехід у директорію
cd rental-car

# Встановлення залежностей
npm install

# Запуск у режимі розробки
npm run dev

# Білд для продакшн
npm run build
npm start
```
